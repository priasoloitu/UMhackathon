UM hackathon project

**JadualIQ** is an AI-powered smart scheduling assistant built for UMHackathon 2026. It uses a multi-agent AI pipeline to help users intelligently plan their week — factoring in weather, traffic, lifestyle restrictions, and real-time conflict detection. It also recommends the best transport modes to save both time and money.
